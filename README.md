 # Personal Website
 Hey :wave: feel free to look around my personal website!
 
**Tech stacks used**:
 * react
 * next.js
 * chakra-ui
 * spotify-api


# Running locally

- Enviornment variables:

```
SPOTIFY_CLIENT_TOKEN=
SPOTIFY_CLIENT_SECRET=
SPOTIFY_REFRESH_TOKEN=
```

- Installing Packages

```
npm install
```
- Running
  - Dev 
    ```
    npm run dev
    ```
  - Production
     ```
     npm run build && npm run start
     ```
